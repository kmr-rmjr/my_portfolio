# Kaito Kimura's Portfolio
https://kaito-kimura.web.app/

# Stack
- Vue.js
- Vuetify
- Fiebase Hosting

# Usage
Clone it.
```
git clone git@github.com:kmrmjr/my_portfolio.git
```

Install dependences.
```
npm install
```

Start server on `localhost:8080` and enjoy.
```
npm run serve
```
